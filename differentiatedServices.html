Simulating Differentiated Services in ns-3

Languages: C++ & XML
Technologies: ns-3, Wireshark
View on GitHub

In this project, I developed a simulation framework for Differentiated Services (DiffServ) using the ns-3 network simulator. The project began with the creation of a base class that supports traffic classification and scheduling, allowing simulation of various QoS mechanisms. The base class was designed to classify packets using custom filters and manage multiple traffic queues based on priority or weight.


I implemented two QoS algorithms:

  • Strict Priority Queuing (SPQ): Packets are transmitted based on assigned priority levels.
  • Deficit Round Robin (DRR): Queues are served fairly using configurable quantum values.

Both mechanisms accept configuration files for flexibility and are validated through custom simulation topologies involving bulk data transfer applications. Performance was evaluated using throughput-over-time plots to demonstrate fairness and priority enforcement.


This project strengthened my understanding of networking concepts and simulation-based validation, and gave me hands-on experience with ns-3 and system-level queueing logic.

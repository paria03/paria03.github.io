Hotel Review Web Application

Languages: Java & JavaScript & HTML & CSS & SQL
Technologies: Jetty, Servlets, JDBC, Fetch API, Bootstrap, MySQL
View on GitHub

This project is a full-stack web application modeled after TripAdvisor, built using Java and Spring Boot for the backend, and HTML/CSS/JavaScript for the frontend. The system loads hotel and review data from JSON files and presents it through a clean, user-friendly web interface.

Core features include:

  • Viewing hotel information and associated reviews
  • Searching for hotels by ID or keyword
  • Organized MVC architecture with clearly separated logic for data loading, processing, and web rendering

This project demonstrates my ability to integrate file-based data with dynamic web content, structure a maintainable backend using Spring Boot, and create responsive, interactive frontend components.

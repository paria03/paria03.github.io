Event Planner Web Application

Languages: Java, HTML, CSS, SQL
Technologies: Spring Boot, Spring Data JPA, MySQL, OpenAI API, Maven
View on GitHub

This project is a full-stack web application that helps users create, manage, and view events. It is built with Spring Boot on the backend and a simple HTML/CSS frontend. Users can register/login, create events, and manage them through a personal dashboard backed by a MySQL database.

Core features include:

  • User authentication with a personalized event dashboard
  • Creating and managing events (date, location, capacity, free vs paid, reservations)
  • Event detail pages displaying full event information
  • AI-powered event description generation using the OpenAI API based on user keywords

This project demonstrates my ability to design backend systems with clean data models and persistence using Spring Data JPA, build user-facing workflows end-to-end, and integrate external APIs to enhance product features.
